== Description ==

Extension permettant de cacher le badge Google ReCaptcha V3 qui se trouve en bas à droite de l'écran.

Il est important de préciser au visiteurs que vous utilisez le service Recaptcha de google, c'est pourquoi vous devez obligatoirement ajouter le texte suivant sur votre site (à la suite de votre formulaire de contact par exemple)

<div id="bscaptcha">
	Ce site est protégé par reCAPTCHA et les <a href="https://policies.google.com/privacy" target="_blank">règles de confidentialité</a> et les <a href="https://policies.google.com/terms" target="_blank">conditions d'utilisation</a> de Google s'appliquent.
</div>
 

== Installation ==

1. Téléchargez l'extension
2. Installez l'extension
3. Activez l'extension
4. ajoutez les informations légales Google

Vous pouvez ajouter l'ID "bscaptcha" pour donner un style aux informations légales qui suivent votre formulaire de contact

Exemple : 
< div id="bscaptcha">
	Ce site est protégé par reCAPTCHA et les < a href="https://policies.google.com/privacy" target="_blank">règles de confidentialité< /a> et les < a href="https://policies.google.com/terms" target="_blank">conditions d'utilisation< /a> de Google s'appliquent.
< /div>
